# HER (Hindsight Experience Replay) 算法

本项目实现了结合 **DDPG** 的 **HER** 算法。为了不依赖复杂的 Mujoco 机器人环境，我们复现了经典的 **BitFlipping** 环境（翻转比特）来演示 HER 在稀疏奖励下的强大威力。

## 目录结构

| 文件 | 说明 |
| :--- | :--- |
| `her_agent.py` | **智能体定义**：修改版的 DDPG，输入包含 Goal。 |
| `her_replay_buffer.py` | **经验回放**：核心组件，包含 HER 采样策略（将失败的经验修改为“成功达成其他目标”的经验）。 |
| `train_her.py` | **训练脚本**：BitFlipping 环境的定义与训练循环。 |

## 核心算法原理

### 1. 稀疏奖励问题
在很多任务中（如穿针引线），除非完全成功，否则奖励全是负的（或0）。普通的 RL 很难随机探索到成功的状态，导致无法学习。

### 2. 后见之明 (Hindsight)
HER 的核心思想是：**即使我不小心走错了路（没达成原定目标 A），但我起码到达了某个地方 B。如果我的目标本来就是 B，那这次尝试不就成功了吗？**

通过将回放池中的 `desired_goal` 替换为实际达成的 `achieved_goal`，并重新计算奖励（改为成功奖励），Agent 能源源不断地获得正反馈，从而学会控制环境的物理规律，最终学会达成任意目标。

### 3. BitFlipping 环境
*   **状态**: N 个 0 或 1。
*   **动作**: 选择翻转第 i 位。
*   **目标**: 变成指定的一串 0/1 序列。
*   **奖励**: 只有完全一致才得 0 分，否则 -1 分。

在 N=10 时，状态空间 $2^{10}=1024$，随机乱撞很难成功。HER 能在几百个 Episode 内迅速解决。

## 使用方法

### 运行训练
```bash
python train_her.py
```
你将看到 Success Rate 迅速从 0.0 上升到 1.0。
