# Weak-to-Strong Generalization (弱到强泛化)

本项目实现了 OpenAI 提出的 **Weak-to-Strong Generalization** 核心逻辑。该技术旨在解决超人工智能（Superalignment）中的对齐问题：即当模型（强模型）比人类或监督者（弱模型）更聪明时，我们如何确保它依然听从人类的意图。

## 目录结构

| 文件 | 说明 |
| :--- | :--- |
| `w2s_agent.py` | **智能体定义**：包含带分类头的模型和专门设计的对齐损失函数。 |
| `train_w2s.py` | **训练脚本**：演示弱模型标注数据引导强模型训练的流程。 |

## 核心算法原理

### 1. 弱模型监督 (Weak Supervision)
在超级智能场景下，人类无法提供完美的标签。我们只能利用现有的二流模型（弱模型）生成的标签来训练更好的模型（强模型）。

### 2. 超越监督者 (Overshooting)
研究证明，强模型在弱标签的引导下，通过特定的正则化和损失函数（如 KL 散度约束、置信度惩罚等），其最终表现可以**显著超越**提供标签的弱模型。

### 3. 损失函数
除了标准的交叉熵，通常还会引入一致性损失，确保强模型不仅学习表面标签，还学习底层的任务逻辑。

## 使用方法

### 运行训练
```bash
python train_w2s.py
```
虽然由于计算资源限制，演示使用的是 GPT-2，但其背后的**弱监督对齐**思想是实现超级对齐（Superalignment）的关键路径。
